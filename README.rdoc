prettyRadBar.js

<i>This little jQuery plugin receives random amount of hex color codes and creates a full width resizable bar.</i>

Example:


<code>
var colours = [ '999999', 'B2E1DB', '308FCF', 'EC1783', '2B9C68' ]; \n
$('#myRadBar').prettyRadBar({ \n
	color: colours, \n
	animate: true \n
});
</code>

Adjust your css as necessary, for example:

<code>
#myRadBar {
	position:relative;
	overflow: hidden;
	width: 100%;
	div {
		height: 12px;
		float: left;
	}
}
</code>

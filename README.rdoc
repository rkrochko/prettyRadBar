This little jQuery plugin receives random amount of hex color codes and creates a full width resizable bar.

Example:

var colours = [ '999999', 'B2E1DB', '308FCF', 'EC1783', '2B9C68' ];
$('#myRadBar').prettyRadBar({
	color: colours, 
	animate: true
});

Adjust your css as necessary, for example:

#myRadBar {
	position:relative;
	overflow: hidden;
	width: 100%;
	div {
		height: 12px;
		float: left;
	}
}
